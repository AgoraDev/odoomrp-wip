MRP lock lot
============
This module allows you to define whether a lot is locked or not. By default,
the lots will be created in locked status.To lock/unlock a lot, have created 2
new buttons.
When you lock one lot, validates that there is no movement of stock with
destination location type virtual/company, have assigned a quant, which has the
lot.
Changes in manufacturing, and delivery orders, to selecting the locked bundles
is prevented.
